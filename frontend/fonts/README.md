# D2Coding 폰트

`styles.css`의 `@font-face`가 이 폴더의 파일을 읽어 글꼴을 적용합니다.

- `D2CODING-VER1.3.2-20180524-ALL.TTC` — 보통(400), 굵게(700) 컬렉션(우선)
- `D2CODINGBOLD-VER1.3.2-20180524.TTF` — 굵게(700) 보조(호환용)

다른 PC에서는 위 파일을 같은 경로(`frontend/fonts/`)에 복사하거나,  
`%LocalAppData%\Microsoft\Windows\Fonts\` 에서 동일 파일을 복사해 넣으면 됩니다.

라이선스는 D2Coding 배포처(네이버 등) 정책을 따릅니다.
